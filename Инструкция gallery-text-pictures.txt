1. Разархивировать архив
2. Скопировать файл с расширением .ggt в папку программы - html (папка с файлами .ggt)
2. Скопировать файл с расширением .ggsk в папку программы - skins (папка с файлами .ggsk)
3. В программе Pano2VR зайти во вкладку Экспорт
4. В программе Pano2VR в строке шаблон: выбрать скапированный .ggsk
5. В программе Pano2VR в окне HTML выбрать скопированный файл .ggt
6. В программе Pano2VR в окне Дополнительные параметры - Ресурсы указать путь к папке pack
7. В программе Pano2VR нажать на шестеренку собрать новый проект

Замена названий:
1)Открыть файл "galery-7-с названиями картинок.html" в редакторе кода
2)Найти заголовок с текстом "<h3>Lorem ipsum dolor 1</h3>"
3)Заменить "Lorem ipsum dolor" на нужное название
4)Выделить весь код и скопировать
5)В программе Pano2VR открыть редактирование шаблона "galery-7-с названиями картинок.ggsk"
6)Нажать на текстовый фрейм и в поле текст открыть логический блок
7)В окне "Текст:" удалить код и вставить скопированный html код
8)Сохранить шаблон, собрать проект

Редактирование стилей текста названия (цвет, размер):
1)Открыть файл css из папки pack-сss-main.css в редакторе кода
2) Найти комментарий "/* Стили для названия картинок */"
3) Заменить значение, сохранить файл, собрать проект
