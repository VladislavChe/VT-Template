1. Достать файлы из архива
2. Закинуть в папку "html5" файл .ggt (html разметка)
3. Закинуть в папку "Skins" файл .ggsk
4. Открыть программу pano2VR
5. Во вкладке "Экспорт" нажать на зеленый + (Создать новый экспорт) и выбрать HTML 
6. Ниже в поле "Шаблон" указать путь до файла .ggsk в папке "Skins"
7. Ниже во вкладке HTML внутри найти поле "Шаблон" и указать путь до файла .ggt в папке "html5"
8. Ниже во вкладке Дополнительные параметры в поле Ресурсы нажать на + и выбрать добавить папку. Добавляем папку "pack"
9. Наверху нажать на "Шестеренку" и собрать проект локально
10. Скопировать содержимое новосозданной папки с проектом на сервер

Настройка галлереи:

1) Изменить высоту картинки:
- Открываем main.css в редакторе кода
- Находим в классе .gallery-slider .slick-track значение  "height: 93%" и комментарий /* Тут меняется высота картинки */ 
- изменяем значение в процентах

2) Изменить расположение точек (буллетов) ПО ВЕРТИКАЛИ:
- Открываем main.css в редакторе кода
- Находим в классе .gallery-slider .slick-dots значение "bottom: 3px;" и комментарий /* Это значение двигает точки вверх вниз */ 
- 3px это расстояние снизу от нижней части фрейма

3) Изменить расположение точек (буллетов) ПО ГОРИЗОНТАЛИ:
- Открываем main.css в редакторе кода
- Находим в классе .gallery-slider .slick-dots значение "left: 50%;" и комментарий /* Расположение точек по горизонтали */
- Указать занчение в процентах

4) Изменить размер точек (буллетов):
- Открываем main.css в редакторе кода
- Находим в классе .gallery-slider .slick-dots button значение "padding: 3px" и комментарий /* Размер точек */ 
- 3px это отступ вовнутрь точки, увеличивая отступ - увеличивается размер точки

5) Настройка скорости анимации слайдов:
- Открываем main.js в редакторе кода
- Находим комментарий /* Скорость анимации */
- Изменяем числовое значение в миллисекундах "speed: 1000"

6) Настройка скорости автопрокрутки слайдов:
- Открываем main.js в редакторе кода
- Находим комментарий /* Скорость автопрокрутки */
- Изменяем числовое значение в миллисекундах "autoplaySpeed: 2500"

Создание градиента фона:

1) Открываем сайт https://www.cssmatic.com/gradient-generator#
2) Выбрать тип градиента (стиль, плавность перехода и т.д.)
3) На левом нижнем ползунке установить начальный цвет
4) На правом нижнем ползунке установить конечный цвет
5) Справа где демонстрационное окно можно выбирать направление перехода градиента
6) Под демонстрационным окном выделить готовый код мышкой и скопировать его
7) Открыть шаблон .ggsk в программе Pano2VR и зайти в редактор шаблона
8) Выбрать нужный элемент и развернуть вкладку расширенные
9) Найти пункт "CSS Стили:" и вставить туда скопированный код из генератора градиента
10) Закрыть и сохранить шаблон
